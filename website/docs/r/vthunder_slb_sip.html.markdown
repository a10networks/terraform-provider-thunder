# vthunder\_slb\_sip

`vthunder_slb_sip` Provides details about vthunder SLB sip
## Example Usage


```hcl

  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_slb_sip" "testname" {
	sampling_enable {
	    counters1 = "all"
	}
}
```
