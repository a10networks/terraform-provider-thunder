# vthunder\_slb\_template\_cipher

`vthunder_slb_template_cipher` Provides details about vthunder SLB template cipher
## Example Usage


```hcl
provider "vthunder" {
  address  = "192.0.2.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_slb_template_cipher" "cipher" {
	cipher_cfg {
		priority = 10
		cipher_suite = "SSL3_RSA_DES_192_CBC3_SHA"
	} 
	name = "testcipher"
	user_tag = "test_user"
}
```
